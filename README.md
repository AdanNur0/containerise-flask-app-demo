# Flask Hello World in Docker

A minimal "Hello World" web app built with Python + Flask, fully containerized with Docker.

Perfect as a first Docker project, learning example, or quick reference for Flask + Docker basics.

## Features
- Super simple Flask application
- Uses modern slim Python base image (3.12-slim)
- Single Dockerfile – no Compose, no extras
- Helper script (`run.sh`) for easy build/run/cleanup
- Step-by-step instructions + troubleshooting tips

## Quick Start (Recommended)

Just run this one command:

```bash
./run.sh all
